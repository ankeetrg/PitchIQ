## Premium Attackers Still Set the Slate

World Cup fantasy starts with the obvious names for a reason. Kylian Mbappe, Vinicius Jr., and Brazil's front line can break a slate without needing 90 perfect minutes.

Pay up when the matchup gives them volume. Fade only when ownership gets reckless or rotation risk is real.

## Value Comes From Roles

The best cheaper plays are not random punts. They take set pieces, start wide against slow fullbacks, or play as late runners into the box.

- Set-piece midfielders carry a safer floor.
- Pressing forwards can hit value with shots and assists.
- Goalkeepers matter most when save volume is clear.

## Differentials Win Large Fields

Low ownership is only useful when the player has a path to points. A winger facing a backup fullback is interesting. A defensive midfielder with no set pieces is not.

PitchIQ flags ownership as an estimate. Treat it as a guide, not a fact.

## Best Build Rule

Stack one strong attack, then find one low-owned player from a match with real goal upside. That gives you ceiling without turning the lineup into a lottery ticket.
